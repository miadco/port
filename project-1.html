These labs focused on managing identities and securing access with Microsoft Entra ID. I configured users, RBAC roles, conditional access, PIM, and SSO for enterprise apps. The work reflects real-world scenarios aligned with SC-300 objectives.

Project Overview

As part of my SC-300 labs, I worked hands-on with Microsoft Entra ID to manage users, groups, and roles. I configured group-based licensing, custom RBAC roles, and conditional access policies with MFA enforcement. I also implemented Privileged Identity Management for just-in-time admin access and set up identity governance features like access packages and reviews. Finally, I configured enterprise apps with SAML-based single sign-on and assigned scoped access to users and groups.
