Project Overview

As part of my AZ-104 learning path, I completed labs designed to simulate the daily responsibilities of an Azure Administrator. I provisioned and managed Azure resources, including virtual machines, storage accounts, and network components. I configured role-based access control and resource locks to apply proper governance across subscriptions. I practiced deploying resources using both the Azure portal and Azure CLI to reinforce infrastructure management flexibility. These labs helped me build a strong foundation in Azure infrastructure, operations, and access control.
