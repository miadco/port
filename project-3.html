Project Overview

I completed hands-on labs focused on securing identities, resources, and workloads in Microsoft Azure. I implemented network security rules, configured just-in-time (JIT) VM access, and applied role-based restrictions using RBAC and PIM. I worked with Microsoft Defender for Cloud to assess security posture and remediate threats, and configured Key Vault for secure key and secret management. I also created custom Azure policies and assignments to enforce compliance across subscriptions. These labs strengthened my understanding of identity protection, resource hardening, and proactive cloud security monitoring.
